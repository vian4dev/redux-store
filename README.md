## ⚙️ Instalação
Clone o repositório.
~~~
git clone https://github.com/vian4dev/redux-store.git
~~~
Acesse o diretório do projeto.
~~~
cd redux-store
~~~
Instale as dependências.
~~~
yarn
~~~
Execute o servidor JSON server.
~~~
yarn server
~~~
Execute a aplicação.
~~~
yarn start
~~~
